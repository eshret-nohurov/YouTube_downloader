# YouTube Загрузчик от Эшрета

Нативное приложение для **macOS** на SwiftUI — удобная графическая обёртка над [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) и [`ffmpeg`](https://ffmpeg.org). Вставьте ссылку на видео YouTube, выберите качество — и файл скачается в нужную папку.

## Возможности

- 🎬 Скачивание видео в разных качествах: «Лучшее», 4K / 2K / 1080p / 720p / 480p / 360p.
- 🎵 Извлечение **только аудио** в MP3.
- 🍎 Приоритет кодеков **H.264 + AAC** для совместимости с QuickTime; форматы AV1/VP9 автоматически перекодируются в MP4.
- ⬇️ Несколько одновременных загрузок с прогрессом, скоростью и оставшимся временем; отмена, повтор, открытие файла и показ в Finder.
- 🔑 Авторизация на YouTube через cookies: экспорт из браузера, готовый файл `cookies.txt` или `--cookies-from-browser`.
- 📦 Автоматическая установка `yt-dlp` и `ffmpeg` — через Homebrew, а при его отсутствии прямой загрузкой.
- 📝 Журнал событий с записью в файл для диагностики.

## Требования

- macOS 13.0 (Ventura) или новее.
- Браузер с авторизацией на YouTube (Chrome, Firefox, Brave, Edge или Safari) — для приватных/возрастных видео и обхода проверок.
- `yt-dlp` и `ffmpeg` — приложение предложит установить их автоматически при первом запуске.

## Установка

### Готовая сборка

Скачайте `.dmg` из раздела [Releases](https://github.com/eshret-nohurov/YouTube_downloader/releases), перетащите приложение в `Applications`.

> Сборка не подписана сертификатом разработчика. При первом запуске macOS может предупредить о неизвестном разработчике: нажмите по приложению правой кнопкой → **Открыть** → **Открыть**.

### Сборка из исходников

```bash
git clone https://github.com/eshret-nohurov/YouTube_downloader.git
cd YouTube_downloader

# Сборка Release и упаковка в .dmg
./build_dmg.sh

# …или обычная сборка через xcodebuild
xcodebuild -project YouTubeLoader.xcodeproj -scheme YouTubeLoader -configuration Release build
```

Можно также открыть `YouTubeLoader.xcodeproj` в Xcode и запустить схему **YouTubeLoader** (⌘R).

## Использование

1. Запустите приложение. При первом старте откроется мастер настройки cookies — экспортируйте их из браузера (откроется окно Terminal, может потребоваться пароль Mac для доступа к keychain).
2. Вставьте ссылку на видео YouTube (при вставке из буфера поиск запускается автоматически).
3. Выберите формат/качество и папку назначения.
4. Нажмите загрузку — прогресс отображается в боковой панели.

**Куда что сохраняется:**

| Что | Путь |
|-----|------|
| Скачанные файлы | папка загрузки (по умолчанию `~/Downloads`, настраивается) |
| Бинарники при прямой установке | `~/Library/Application Support/EshretYTDownloader/bin` |
| Экспортированные cookies | `~/Library/Application Support/EshretYTDownloader/cookies.txt` |
| Журнал событий | `~/Library/Logs/YouTubeLoader/log_<дата>.txt` |

## Как это работает

Приложение построено по MVVM (`Views` / `ViewModels` / `Models` / `Services`). Вся работа с YouTube сосредоточена в `YtDlpService`, который формирует аргументы `yt-dlp`, запускает его и парсит вывод; `DownloadManager` управляет очередью загрузок, `InstallerService` ставит зависимости, `TelemetryService` ведёт журнал.

## Зависимости

- [yt-dlp](https://github.com/yt-dlp/yt-dlp) — загрузка с YouTube.
- [ffmpeg](https://ffmpeg.org) — объединение и перекодирование дорожек.

Обе утилиты устанавливаются приложением автоматически и не входят в репозиторий.
